## 文档 1：饮品标准配方（v3.2）

**标题**：常规饮品制作规范（含容差、替代方案与客制化指南）

### 美式咖啡（Americano）
- **标准配方**：双份浓缩（约36ml，误差±2ml） + 热水至240ml（水温严格控制在92±1℃）。  
- **为什么是92℃？**：过高会萃取焦苦味，过低则酸涩突出；店内恒温热水机已预设此值，勿手动调节。  
- **客制化选项**：  
  - 少水：可做至200ml（需在POS备注“少水美式”）  
  - 冰美式变体：见“冰美式”条目，不可混用热水  
- **常见错误**：  
  - ❌ 先加水再倒浓缩 → 导致油脂（crema）消散，风味扁平  
  - ✅ 正确顺序：先萃取浓缩，再缓慢注入热水  

### 拿铁（Latte）
- **标准配方**：双份浓缩 + 蒸汽打发全脂牛奶200ml（奶泡厚度≤0.5cm，用尺卡测量）。  
- **牛奶要求**：必须使用“本地乳业”全脂奶（脂肪含量≥3.5%），脱脂奶禁止用于拿铁（无法形成稳定奶泡）。  
- **打发技巧**：  
  - 蒸汽棒插入液面下1cm，先“嘶嘶”进气1秒，再沉入加热  
  - 温度达60℃时停止（手感烫但可握3秒）  
- **视觉验收**：  
  - 倾倒后表面光滑如镜，无大气泡  
  - 若用于拉花，需在30秒内完成（否则奶泡塌陷）  
- **例外情况**：  
  - 顾客要求“厚奶泡拿铁” → 实际应推荐卡布奇诺，并解释区别  

### 卡布奇诺（Cappuccino）
- **标准配方**：单份浓缩（18ml） + 打发牛奶150ml，奶泡厚度1.0–1.5cm（可用专用量勺压测）。  
- **为什么奶泡更厚？**：传统意式比例为1:1:1（浓缩:液体奶:奶泡），我店微调为1:0.8:0.7以适应本地口味。  
- **装饰规范**：  
  - 可撒肉桂粉（仅限堂食，外带易洒）  
  - 禁止用巧克力粉（属摩卡系列）  
- **顾客误解应对**：  
  > 问：“为什么比隔壁店泡沫少？”  
  > 答：“我们追求奶泡绵密而非蓬松，这样口感更顺滑，您试试看？”

### 摩卡（Mocha）
- **标准配方**：单份浓缩 + 巧克力酱15g（用电子秤校准，误差±1g） + 蒸奶180ml，顶部加鲜奶油（仅堂食提供）。  
- **巧克力酱管理**：  
  - 开封后冷藏，保质期7天（标签注明开封日）  
  - 冬季需提前10分钟取出回温，否则难以泵出  
- **鲜奶油操作**：  
  - 使用N₂O气弹奶油枪，压力0.8MPa  
  - 挤出高度2cm，呈螺旋状  
  - 外带订单自动替换为“额外巧克力酱5g”，并在小票标注“无奶油（外带）”  
- **过敏提示**：  
  - 巧克力酱含大豆卵磷脂，对大豆过敏者需提前告知  

### 冰美式（Iced Americano）
- **标准配方**：双份浓缩 + 冰块120g（约8块标准立方体） + 冷水至杯满（总容量约300ml）。  
- **冰块标准**：  
  - 使用制冰机“透明冰”模式（非碎冰）  
  - 每日早班检查冰桶清洁度，霉斑立即报废  
- **稀释控制**：  
  - 浓缩需直接淋在冰上（非先加水），利用冰融控制浓度  
  - 若顾客要求“少冰”，则冰减至80g，水补至300ml  
- **夏季特别提示**（6–8月）：  
  - 可提供“双倍浓缩冰美式”（+¥3），应对高温稀释加速  

> **通用注**：所有浓缩均使用店内 La Marzocco Linea Mini 意式咖啡机，研磨度设为“中细”（刻度4.5/10），萃取时间25–30秒。若环境湿度>70%，研磨度需调细0.5档（记录于《每日参数表》）。

## 文档 2：员工排班与考勤制度（v2.7）

**标题**：工作时间、请假、调班与异常处理全流程

### 营业时间与班次定义
- **标准营业**：每日 7:00 – 20:00（全年无休，除春节初一）  
- **早班**：7:00–15:00（含30分钟轮休，不可连续工作超5小时）  
- **晚班**：12:00–20:00（含30分钟轮休，高峰时段12–14点不得离岗）  
- **重叠时段**（12:00–15:00）：必须保证至少3人在岗（1收银+1制作+1清洁/打包）  

### 排班规则
- **每周工作5天**：连续工作≤6天，必须安排1天完整休息  
- **排班提交**：  
  - 每周五18:00前在共享表格填写下周“可上班时段”  
  - 格式：`3/24(Mon): 早, 晚 | 3/25(Tue): 早`  
  - 未填写者，默认接受任意排班  
- **公平原则**：  
  - 周末/节假日班次轮流分配，新员工首月不排高峰日  
  - 生日当月可优先选择一天 off（需提前14天申请）  

### 请假流程
- **临时病假**：  
  - 需至少提前4小时电话通知店长 + 企业微信提交申请  
  - 当日提供医院证明（如发烧需体温记录）  
  - 无证明按事假处理，扣当日工资  
- **事假**：  
  - 提前72小时申请，获批后方可离岗  
  - 每月事假≤2天，超限需店长特批  
- **紧急情况**（如直系亲属病危）：  
  - 可事后补交证明，不扣工资，但需提供警方/医院文件  

### 迟到与旷工
- **迟到分级**：  
  - 0–15分钟：扣当日工资10%  
  - 16–30分钟：扣30%  
  - >30分钟：视为旷工半天，扣全天工资  
- **月累计**：  
  - 迟到≥3次：书面警告  
  - 旷工1次：停岗培训1天  
  - 旷工2次：解除合同  
- **豁免情形**：  
  - 地铁故障（需提供APP延误证明）  
  - 暴雨红色预警（以气象局公告为准）  

> **注意**：打卡以企业微信定位为准（门店50米内），代打卡双方均视为严重违纪。

## 文档 3：设备维护日志模板（v4.1）

**标题**：咖啡机、磨豆机、净水系统全周期保养手册

### 每日操作（早/晚班各执行一次）
- **开机前**（早班）：  
  1. 清空渣盒（称重记录，>200g 需报告）  
  2. 冲洗冲煮头3次（每次5秒，观察水流均匀性）  
  3. 检查锅炉压力表（0.8–1.0 bar 为正常）  
- **每4小时**（所有班次）：  
  - 用盲碗+ Cafiza 清洁粉进行反冲洗（backflush）  
  - 步骤：装盲碗 → 加1勺粉 → 萃取10秒 → 停5秒 → 重复3次 → 清水冲洗2次  
  - 记录于《反冲洗日志》，含操作人、时间、异常描述  
- **打烊后**（晚班）：  
  1. 排空锅炉余水（打开蒸汽阀10秒）  
  2. 关闭主电源（非待机键）  
  3. 清洁磨豆机：  
     - 清空豆仓，用软毛刷清理刀盘残粉  
     - 用 Grindz 专用清洁豆运行30秒（每周一、四执行）  

### 每周维护（店长执行，周日晚班）
- **净水器滤芯**：  
  - 型号：3M Aqua-Pure AP11T  
  - 更换周期：每4周（或流量计达5000L）  
  - 记录：旧滤芯拍照存档，新滤芯序列号录入系统  
- **蒸汽棒疏通**：  
  - 用0.7mm 专用针从出口向内轻捅（勿用回形针！）  
  - 测试：空喷蒸汽3秒，观察是否呈直线（非散射）  
- **磨豆机校准**：  
  - 用18g 咖啡粉测试萃取时间（目标28秒）  
  - 偏差>2秒则微调研磨度  

### 故障报修流程
1. 初步排查：  
   - 无压力？→ 检查水源阀门  
   - 萃取慢？→ 清洁冲煮头  
2. 仍无效 → 拍照/视频发张工微信  
3. 张工响应：  
   - 工作日2小时内到场  
   - 非工作日收¥200 加急费  
4. 维修期间：  
   - 启用备用 Aeropress 制作美式/拿铁（限5杯/小时）

## 文档 4：顾客投诉处理指南（v5.0）

**标题**：服务异常应对、补偿标准与法律边界

### 四步法（必须按顺序执行）
1. **倾听**：  
   - 保持眼神接触，身体前倾15度  
   - 不打断，不辩解，用“嗯”“我理解”回应  
   - 时长≤3分钟，超时请店长介入  
2. **道歉**：  
   - 标准话术：“非常抱歉给您带来不便！”（即使非我方责任）  
   - 禁用：“但是…”“因为…”“别人也…”  
3. **解决**（按问题类型）：  
   - **饮品做错**：  
     - 立即重做（优先级最高）  
     - 原饮品收回销毁（不可二次使用）  
     - 赠送小饼干（成本¥0.8/块）  
   - **等待过久**（>15分钟）：  
     - 赠送“下次饮品9折券”（系统生成，7天有效）  
     - 若顾客拒绝，可升级为“免费小杯美式”  
   - **卫生问题**（如杯具污渍）：  
     - 店长出面，全额退款  
     - 赠送“道歉券”（限下次使用，不可兑换现金）  
     - 24小时内提交整改报告  
4. **记录**：  
   - 填写《客诉登记表》（纸质+电子）  
   - 字段：时间、顾客特征、问题描述、处理人、补偿方式、后续改进  
   - 24小时内店长审核归档  

### 补偿上限与权限
| 问题类型       | 员工可决策 | 需店长批准 |
|----------------|------------|------------|
| 饮品重做       | ✓          | —          |
| 赠送小饼干     | ✓          | —          |
| 9折券          | ✓          | —          |
| 全额退款       | —          | ✓          |
| 赠送正价饮品   | —          | ✓          |

### 法律红线
- **绝不承诺**：“下次一定…”“绝对不会再…”  
- **绝不承认**：“我们卫生不合格” → 改为“我们会加强检查”  
- **录像调取**：仅店长可申请查看监控，且需顾客书面同意  

> **案例**：2026年2月，顾客称“喝出虫子”，实为咖啡渣结块。因员工说“不可能有虫”，引发舆情。正确做法：先道歉，再邀请第三方检测。

## 文档 6：卫生与食品安全规范（v4.3）

**标题**：员工个人卫生、环境清洁与食安红线（含 HACCP 要点）

### 员工个人卫生（强制执行）
- **洗手要求**：  
  - 时机：上岗前、如厕后、处理垃圾后、接触生食/现金后  
  - 方法：使用抗菌洗手液（含氯己定），搓洗≥20秒（默念两遍“咖啡拿铁美式摩卡”）  
  - 干燥：仅用一次性擦手纸（禁止甩手或用围裙）  
- **着装规范**：  
  - 围裙：每日更换，污渍>3cm² 立即更换  
  - 头发：必须完全束起 + 戴网帽（刘海不可外露）  
  - 首饰：禁止戒指、手链、耳坠（医用耳钉可保留）  
  - 指甲：长度≤1mm，无指甲油、美甲  
- **健康状态**：  
  - 出现腹泻、呕吐、黄疸、发烧（≥37.5℃） → 立即离岗，持医院康复证明返岗  
  - 皮肤伤口：必须覆盖防水创可贴 + 戴手套（双层）

## 文档 7：季节性菜单（春季示例 · v2.9）

**标题**：2026年3–5月限定饮品全操作手册（含客诉预案）

### 樱花拿铁（热/冰）
- **配方**：  
  - 浓缩：双份（36ml）  
  - 樱花糖浆：10ml（泵头校准：1泵=5ml）  
  - 牛奶：200ml（全脂，60℃打发）  
  - 装饰：食用樱花碎0.1g（镊子夹取，勿手触）  
- **关键控制点**：  
  - 糖浆批次：仅限“樱之味”品牌（批号 YH202602XX）  
  - 樱花碎保质期：开封后7天（冷藏，避光）  
- **客制化**：  
  - 少糖：糖浆减至5ml（不可为0，否则风味失衡）  
  - 燕麦奶替代：+¥2，需告知“樱花风味略弱”  
- **客诉预案**：  
  > 问：“糖浆是香精吧？”  
  > 答：“我们使用天然樱花萃取糖浆，配料表可查，您要看看吗？”（递上供应商检测报告复印件）

### 抹茶椰乳
- **配方**：  
  - 抹茶粉：5g（宇治五十铃，过筛防结块）  
  - 椰乳：150ml（Aroy-D 品牌，摇匀后开罐）  
  - 冰块：100g  
  - 牛奶：补至杯满（约100ml）  
- **操作难点**：
