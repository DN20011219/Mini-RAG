# 咖啡店内部文档（2026年版）

---

## 文档 1：饮品标准配方

**标题**：常规饮品制作规范  

- **美式咖啡**：双份浓缩（约36ml） + 热水至240ml（水温92℃）。  
- **拿铁**：双份浓缩 + 蒸汽打发全脂牛奶200ml，奶泡厚度≤0.5cm。  
- **卡布奇诺**：单份浓缩 + 打发牛奶150ml，奶泡较厚（约1–1.5cm），表面可撒肉桂粉。  
- **摩卡**：单份浓缩 + 巧克力酱15g + 蒸奶180ml，顶部加鲜奶油（仅堂食提供）。  
- **冰美式**：双份浓缩 + 冰块 + 冷水至杯满（约300ml）。  

> 注：所有浓缩均使用店内标准意式咖啡机，研磨度设为“中细”，萃取时间25–30秒。

## 文档 2：员工排班与考勤制度

**标题**：工作时间与请假流程  

- 营业时间：每日 7:00 – 20:00，分早班（7:00–15:00）与晚班（12:00–20:00）。  
- 员工每周工作5天，需提前一周在排班表上标注可上班时段。  
- 临时请假须至少提前4小时通知店长，并在企业微信提交请假申请。  
- 迟到15分钟以内扣当日工资10%，超过30分钟视为旷工半天。

## 文档 3：设备维护日志模板

**标题**：咖啡机与磨豆机日常保养  

**每日操作**：  
- 开机前：清空渣盒，冲洗冲煮头3次。  
- 每4小时：用盲碗+清洁粉进行反冲洗（backflush）。  
- 打烊后：排空锅炉余水，关闭电源，清洁磨豆机豆仓。  

**每周维护（由店长执行）**：  
- 更换净水器滤芯（每4周一次，记录于《耗材更换表》）。  
- 检查蒸汽棒是否堵塞，用专用针疏通。  

> 故障报修联系人：张工 138****1234（设备供应商）

## 文档 4：顾客投诉处理指南

**标题**：服务异常应对流程  

1. **倾听**：保持微笑，不打断顾客陈述。  
2. **道歉**：即使非我方责任，也先说“非常抱歉给您带来不便”。  
3. **解决**：  
   - 饮品做错 → 立即重做，原饮品收回。  
   - 等待过久 → 赠送小饼干或下次饮品9折券。  
   - 卫生问题 → 店长出面，全额退款并赠送道歉券（限下次使用）。  
4. **记录**：填写《客诉登记表》，24小时内提交店长归档。

## 文档 6：卫生与食品安全规范

**标题**：员工个人卫生要求  

- 上岗前必须洗手（使用抗菌洗手液，搓洗≥20秒）。  
- 工作期间不得佩戴首饰、留长指甲或涂指甲油。  
- 如有咳嗽、发烧、腹泻等症状，禁止接触食品，立即调岗或休息。  
- 围裙每日更换，头发需完全束起并佩戴帽子。

## 文档 7：季节性菜单（春季示例）

**标题**：2026年3–5月限定饮品  

- **樱花拿铁**（热/冰）：浓缩 + 樱花糖浆10ml + 牛奶200ml，顶部撒食用樱花碎。  
- **抹茶椰乳**：抹茶粉5g + 椰乳150ml + 冰块，摇匀后加牛奶至满杯。  
- **姜黄燕麦拿铁**：姜黄粉1g + 燕麦奶200ml + 浓缩，推荐给注重养生的顾客。  

> 限定饮品需在收银系统中手动选择“春季特调”分类。
